# Data Engineering E-commerce Platform

## Objective
Design and implement an end-to-end data engineering platform to ingest, transform,
model, and analyze e-commerce sales data using open-source tools.

## Tech Stack
- Python
- Apache Spark (PySpark)
- Apache Airflow
- PostgreSQL
- dbt Core
- Power BI Desktop
- Docker & Docker Compose
